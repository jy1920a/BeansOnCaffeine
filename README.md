BeansOnCaffeine is a lightweight static web project focused on AI-driven automation, productivity, and business workflow optimization.

The project explores how AI can be applied in real-world scenarios, especially within ERP systems, logistics, and operational processes.

- Showcase ideas around productivity and efficiency
- Serve as a foundation for future AI tools and integrations
